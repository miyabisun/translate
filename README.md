# translate

## Installation

require Node.js

```bash
$ npm install -g miyabisun/translate
```

### set auth-file

- `~/.config/.deepl.json`
- `~/.deepl.json`

```bash
$ cat ~/.config/.deepl.json
{
  "free_api": true,
  "auth_key": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx:fx"
}
```

## Usage

These commands will be available

- `2en <message>`
- `2jp <message>`
